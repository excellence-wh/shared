Shared Scripts Documentation

用于构建、清理、恢复和压缩项目的共享脚本集合

Wuhan Excellence Technology Co., Ltd.

脚本目录

clean.ts build.ts restore.ts publish.ts zip.ts

概述

本目录包含一系列用于项目管理的共享脚本，支持Node.js和.NET项目。这些脚本旨在简化项目构建、清理、依赖恢复和发布过程，提高开发效率。

使用方法

所有脚本都可以通过以下方式运行：

bun run <script-name>

或者在其他项目中引用：

bun run d:/excellence.wh/shared/scripts/src/<script-name>.ts

clean.ts

功能

根据项目配置清除指定的文件和目录

状态

已实现

项目类型

Node.js, .NET

使用方法

bun run clean [options]

参数说明

参数 别名 类型 描述 默认值
--directories -d array 目标扫描目录 ["."]
--dry-run -n boolean 预览操作而不实际执行命令 false
--concurrency -c number 最大并发操作数 5
--verbose -v boolean 启用详细日志 false

配置示例

在项目的package.json中添加clean配置：

{"clean": {
  "files": [
    "dist",
    "*.log",
    "node_modules"
  ]
}}

示例

基本使用

bun run clean

预览模式

bun run clean --dry-run

指定目录

bun run clean --directories ./src ./test

build.ts

功能

构建Node.js和.NET项目

状态

已实现

项目类型

Node.js, .NET

使用方法

bun run build [options]

参数说明

参数 别名 类型 描述 默认值
--directories -d array 目标扫描目录 ["."]
--dry-run -n boolean 预览操作而不实际执行命令 false
--concurrency -c number 最大并发操作数 5
--verbose -v boolean 启用详细日志 false
--configuration --config string .NET项目构建配置 Release

示例

基本使用

bun run build

使用Debug配置构建.NET项目

bun run build --config Debug

使用yarn构建Node.js项目

bun run build --nc "yarn build"

zip.ts

功能

压缩项目输出文件

状态

已实现

项目类型

Node.js, .NET

使用方法

bun run zip [options]

参数说明

参数 别名 类型 描述 默认值
--directories -d array 目标扫描目录 ["."]
--dry-run -n boolean 预览操作而不实际执行命令 false
--concurrency -c number 最大并发操作数 5
--verbose -v boolean 启用详细日志 false
--format -f string 压缩格式 zip
--output -o string 输出文件名模板 {project}-{timestamp}.{format}
--config --conf string 自定义配置文件路径 -

压缩格式

  • zip：标准ZIP格式
  • tar：TAR归档格式
  • tgz：压缩的TAR格式（gzip）

输出文件名模板

支持以下变量：

  • {project}：项目名称
  • {timestamp}：时间戳
  • {format}：压缩格式

示例

基本使用

bun run zip

使用tar格式

bun run zip -f tar

自定义输出文件名

bun run zip -o "{project}-{timestamp}.{format}"

配置示例

在其他项目中使用这些脚本的完整配置示例：

{"name": "my-project",
  "version": "1.0.0",
  "scripts": {
    "clean": "bun run d:/excellence.wh/shared/scripts/src/clean.ts",
    "build": "bun run d:/excellence.wh/shared/scripts/src/build.ts",
    "restore": "bun run d:/excellence.wh/shared/scripts/src/restore.ts",
    "zip": "bun run d:/excellence.wh/shared/scripts/src/zip.ts"
  },
  "clean": {
    "files": [
      "dist",
      "*.log",
      "node_modules"
    ]
  }
}
